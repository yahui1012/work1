內洞國家森林遊樂區

新店青立方休閒農場

看過宮崎駿動畫「天空之城」嗎？位於新店的青立方休閒農場就有宛如天空之城般的美景！青立方是一間咖啡廳，內部裝潢合併中西式風格，而咖啡廳外有一小片紅磚牆以及裝置藝術，紅磚牆的打造就像是天空之城的美景。是一個可以吃東西休息、又可以室內戶外都拍照的好地方，如果覺得只去青立方不夠的話，也可以到附近的「銀河洞瀑布」或是「貓空」走走喔！

新店銀河洞越嶺步道

銀河洞位於新北市新店區，銀河洞瀑布分上下兩段，中有一內凹的山壁所形成的山洞，站在水濂洞內，迎面而來的清風夾帶水氣，相當涼爽。洞內建有一座供奉觀世音菩薩的小廟，山壁之間垂下的蕨類植物襯托嵌於崖壁內的宮廟，煞是好看。
